# Bistro Aurora – Beispiel-Website

Dies ist eine statische Beispiel-Website für das fiktive Restaurant **Bistro Aurora** in Dortmund.  
Sie dient als Design- und Webentwicklungsprojekt.

## Inhalt

- `index.html` – Startseite
- `speisekarte.html` – Speisekarte
- `galerie.html` – Galerie
- `kontakt.html` – Kontakt & Reservierung
- `impressum.html` – Rechtliche Angaben
- `datenschutz.html` – Datenschutzerklärung
- `/css/style.css` – zentrales Stylesheet
- `/img/` – Bilder & Assets

## Features

- Responsives Layout
- Saubere HTML5-Struktur
- Einheitliches CSS-Design
- Vollständige rechtliche Pflichtseiten (Impressum + Datenschutz)

## Lizenz

Dieses Projekt ist als Beispielprojekt veröffentlicht.  
Die Inhalte (Code, Layout, Struktur) dürfen für Lernzwecke frei verwendet werden.

Bildmaterial ist abhängig von den jeweiligen Quellen und darf **nicht** ohne eigenes Nutzungsrecht weiterverwendet werden.
